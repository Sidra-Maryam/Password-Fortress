# Password Fortress

**Password Fortress** is an enterprise-grade password strength analyzer and generator designed to help users create, manage, and evaluate secure passwords. It provides a detailed security score, entropy analysis, and actionable recommendations to strengthen your passwords.

---

## Features

- **Password Strength Analysis**:  
  - Evaluates passwords based on length, uppercase/lowercase letters, numbers, special characters, common password usage, and sequential patterns.
  - Displays a **security score**, **entropy bits**, **character count**, and estimated **time to crack**.

- **Detailed Recommendations**:  
  - Minimum length: at least 12 characters recommended  
  - Uppercase letters (A-Z)  
  - Lowercase letters (a-z)  
  - Numbers (0-9)  
  - Special characters (!@#$%^&*)  
  - Avoid common passwords  
  - Avoid sequential patterns (e.g., 123, abc, qwerty)

- **Generate Strong Passwords**:  
  - Automatically create strong, random passwords based on your criteria.  
  - Ensures all recommended strength parameters are included.

- **User-Friendly Interface**:  
  - Enter a password and instantly see the evaluation results.  
  - Easy-to-understand visual feedback on password strength.

---

## Usage

1. Open the application.  
2. Enter the password you want to evaluate in the input box.  
3. View the security score, entropy, character count, and time to crack.  
4. Review the recommendations to improve password strength.  
5. Use the **Generate Password** feature to create a strong password automatically.
